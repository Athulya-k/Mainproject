AUGMENTED REALITY BASED INDOOR NAVIGATION SYSTEM

Mobile applications based on GPS such as Google Maps have proven to be very useful in guiding us to our destination. However, it is still easy to get lost indoors, where GPS satellite signals are not accurately trackable for navigation applications. Thus introduced the concept of using augmented reality with the aim of creating a navigation system that can help people orient themselves and navigate within large buildings.
